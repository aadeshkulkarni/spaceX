# SpaceX Program

### Installation Steps:
1. Download zip file and unzip
2. Open Terminal inside the unzipped folder and type the following commands:
3. npm install
4. npm run start

### Features:
1. Fetches data from SpaceX API
2. Filter capability
3. Responsive Designs
4. Built using React + Vanilla CSS
